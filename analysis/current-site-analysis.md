---

## 1. Технологична платформа

- **CMS:** WordPress 6.9.4
- **Тема:** Avada (ThemeFusion) — премиум шаблонна тема
- **Плъгини:** WPML (многоезичност), LayerSlider 7.14.0, Google Site Kit
- **JS:** jQuery, Workbox v7.3.0 (Service Worker)
- **Аналитика:** Google Analytics (G-PS67DMR7KR)

## 7. Идентифицирани проблеми

### Критични
1. HTTP линкове към поддомейни (ws.naval-acad.bg, projects.nvna.eu, studentcouncil.naval-acad.bg)
2. Неактуална английска версия

### Средни
3. Разпръсната инфраструктура — множество отделни платформи с различен дизайн
4. Непълна ARIA имплементация
5. LayerSlider забавя зареждането
6. Inline CSS стилове за конкретни page ID-та

### Положителни практики
- Декларация за достъпност
- Банкова информация във футъра
- Добре структурирана страница за специалности с филтри
- Социални мрежи интеграция (Facebook, X, Instagram, LinkedIn)
