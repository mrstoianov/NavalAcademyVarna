6. Идентифицирани проблеми

Критични

HTTP линкове: ws.naval-acad.bg, projects.nvna.eu, studentcouncil.naval-acad.bg — сериозен проблем за сигурността
Неактуална английска версия: Новините на английски изостават с години спрямо българските

Средни

Разпръсната инфраструктура: 9+ отделни портала с различен дизайн и UX
Достъпност: Непълна ARIA имплементация, липса на WCAG 2.1 съответствие
Производителност: LayerSlider забавя зареждането; inline CSS за конкретни page ID-та

Положителни практики

Декларация за достъпност, банкова информация за студенти, филтри за специалности, социални мрежи
